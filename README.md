# SMPCP

## Description

**Smpcp** is a simple client for [Music Player Daemon](https://www.musicpd.org) written in Bash, that includes some more advanced features like:

*  Music non stop.
*  Song rating.
*  Playback statistics.

# Dependences

bash coreutils libmpdclient libnotify util-linux
