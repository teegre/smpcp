# SMPCP

## Description

**Smpcp** is a simple client for [Music Player Daemon](https://www.musicpd.org) written in Bash (and C), that includes some more advanced features like:

*  Music non stop.
*  Song rating.
*  Playback statistics.
*  Voice over.

# Dependences

bash coreutils gnu-netcat libmpdclient libnotify util-linux  
optional: espeak
